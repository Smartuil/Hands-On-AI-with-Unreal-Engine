# 代理人意识

你可以跑，但躲不掉！&#x20;

哦，你回来了？太好了，因为这意味着你的眼睛正在捕捉一些光线信息，你的大脑正在感知这一行为，通常称为阅读。 我们所做的一切，每一个决定都是基于我们所感知的，从生物学上讲，我们做出快速决策过程是因为时间至关重要（例如，你看到一条蛇，你的杏仁核处理这些信息的速度比你的视觉皮层要快得多！）。&#x20;

同样的概念，人工智能需要通过收集它们首先需要感知的信息来基于事实做出决策。这一章都是关于感知，以及人工智能如何从环境中获取这些信息，以便它能够意识到周围的环境。我们在上一章中看了EQS，它收集了大量关于周围环境和处理的信息。在这里，我们将仅限于简单的感知行为。人工智能将如何使用这些信息是其他章节的主题（其中一些我们已经讨论过）。

以下是我们在本章中将面临的主题的快速概述：&#x20;

* 现有视频游戏中的感知和意识
* 虚幻引擎内感测系统概述
* 感知组件
* 视觉和听觉的感官
* 感知刺激
* 在代理中实现视觉（蓝图和C++）
* 在代理中实现听觉（蓝图和C++）

那么，让我们先来看一些视频游戏中人工智能意识的例子，然后我们将看到如何在虚幻引擎中设置一个感知系统。
